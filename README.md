# Nouwe Bridge

Nouwe Bridge is a cross-chain bridge utilizing open intents and Hyperlane, enabling seamless asset transfers between multiple testnets.

## Supported Chains
- **Cappucino Testnet** (Arbitrum Nitro Rollup, Chain ID: 55214)
- **Arbitrum Sepolia**
- **Base Sepolia**
- **Holesky**
- **Sepolia**
- **Optimism Sepolia**
- **Sonic Blaze Testnet**
